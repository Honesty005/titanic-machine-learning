This project is based on the Kaggle Titanic competition.  
The objective is to predict passenger survival using machine learning techniques.

## Overview
The dataset contains information on Titanic passengers such as gender, passenger class, and family size.  
A Random Forest model was trained to classify whether a passenger survived.

## Workflow
- Data loading and preprocessing
- Exploratory data analysis
- Feature selection and encoding
- Model training using Random Forest
- Prediction and submission to Kaggle

## Tools Used
- Python
- Pandas
- Scikit-learn

## Results
Public Kaggle score: **0.77511**
