# Titanic - Machine Learning from Disaster

This project is based on the Kaggle Titanic competition.
The goal is to predict passenger survival using machine learning.

## Steps
- Data cleaning and preprocessing
- Feature engineering
- Model training using Logistic Regression
- Prediction and submission to Kaggle

## Tools
- Python
- Pandas
- Scikit-learn

## Results
Public Kaggle score: 0.78
